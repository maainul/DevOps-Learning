###  If you want Jenkins to **clone your GitHub repo**, update your pipeline like this:

```groovy
pipeline {
    agent any

    stages {
        stage('Clone Repo') {
            steps {
                git url: 'https://github.com/your-username/your-repo.git', branch: 'main'
            }
        }
        stage('Build') {
            steps {
                echo 'Repository cloned. Listing files...'
                sh 'ls -la'
            }
        }
    }
}
```
